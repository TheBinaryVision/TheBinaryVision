### Featured Engineering Projects

#### 1. Prashi Care (Healthcare Platform)
A comprehensive, digital-first healthcare web application designed to streamline medical services and patient care management. 
*   **Architecture:** Built utilizing a modern component-based frontend and a robust backend API structure to ensure secure data handling.
*   **Deployment:** Successfully deployed utilizing Cloudflare and Netlify for optimized performance and reliable uptime.
*   **Impact:** Solves real-world healthcare management challenges by providing accessible digital infrastructure.

#### 2. KrazyStore (E-Commerce Ecosystem)
A dynamic full-stack e-commerce application providing a seamless shopping experience from product discovery to secure checkout.
*   **Capabilities:** Features complex state management, secure user authentication, dynamic routing, and an intuitive UI.
*   **Engineering:** Optimized for high performance, smooth cross-device responsiveness, and scalable product databases.

#### 3. Study Tracker (formerly CareerTracker)
A sophisticated application engineered as a Progressive Web App (PWA), tailored for users to manage learning goals and track career progression.
*   **Capabilities:** Features offline functionality, mobile-first design principles, and seamless cross-platform installation directly to user devices.
*   **Engineering:** Built with advanced caching strategies and service workers to ensure reliable performance on varied network conditions.
